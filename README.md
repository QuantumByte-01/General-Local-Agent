# General-Local-Agent

## Getting started 

Follow these steps to install the `uv` tool (if not already installed), create a virtual environment using `uv`, activate it, and sync dependencies.

1. Install `uv` (if not installed)(https://docs.astral.sh/uv/getting-started/installation/):

```powershell
pip install uv
```

2. Create a virtual environment using `uv`:

```powershell
uv venv
```

3. Activate the virtual environment (PowerShell):

```powershell
.\.venv\Scripts\Activate
```

4. Sync the project environment/dependencies using `uv`:

```powershell
uv sync
```
